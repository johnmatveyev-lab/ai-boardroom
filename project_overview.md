# Project Overview: AI Boardroom (Draft Name: boardme.ai)

## 1. High-Level Vision
AI Boardroom is a multi-agent corporate simulation and execution platform. It allows a human user to act as the ultimate stakeholder, interacting seamlessly with a central interface agent ("Jarvis") via voice or chat. Jarvis manages a "Board of Directors"—a team of highly specialized AI agents that collaborate to build products, generate MVPs, and execute complex workflows.

## 2. Core Interaction Loop
1. **User Input:** The human provides a high-level directive (e.g., "Generate an MVP for a new app").
2. **Intake & Delegation (Jarvis):** Jarvis receives the input, clarifies requirements if necessary, and translates the directive into actionable sub-tasks.
3. **Execution (The Board):** Jarvis assigns the sub-tasks to the appropriate Board Members (CEO, CTO, CMO, etc.).
4. **Collaboration:** The Board Members use specialized skills and tools to complete their tasks, sharing memory and context.
5. **Delivery:** Jarvis compiles the Board's work and presents the final output to the user.

## 3. Development Environment
- **Primary IDE:** Google Ecosystem (e.g., Project IDX or similar cloud IDEs).
- **Goal:** Fast, reliable, and integrated development utilizing the full suite of Google's deployment and building tools.
